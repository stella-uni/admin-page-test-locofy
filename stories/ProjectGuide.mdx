## 🛠️ 시작하기

### 1️⃣ 사전 요구사항

<div style={{ 
  backgroundColor: '#ffe4a3', 
  border: '1px solid #fca311', 
  borderRadius: '8px', 
  padding: '16px', 
  marginTop: '16px',
  marginBottom: '16px'
}}>
  <strong>⚠️ 필수:</strong> Node.js 버전 20 이상이 필요합니다.
</div>

### 2️⃣ 설치

```bash
# 의존성 설치
npm install
```

### 3️⃣ 개발 서버 실행

```bash
# Next.js 개발 서버 실행 (포트 3000)
npm run dev

# Storybook 실행 (포트 6006)
npm run storybook
```

### 4️⃣ 빌드

```bash
# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 📁 프로젝트 구조

```
project/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # 루트 레이아웃
│   ├── page.tsx           # 메인 페이지
│   ├── global.css         # 전역 스타일
│   └── antd.css           # Ant Design 스타일
│
├── components/            # 재사용 가능한 컴포넌트
│   ├── ui/               # UI 컴포넌트
│   │   ├── header.tsx
│   │   ├── left-navigation.tsx
│   │   └── left-navigation-set.tsx
│   ├── kpi-card.tsx      # KPI 카드
│   ├── search-bar.tsx    # 검색 바
│   └── chip.tsx          # 칩 컴포넌트
│
├── stories/              # Storybook 스토리
│   ├── components/       # 컴포넌트 스토리
│   └── design-tokens/    # 디자인 토큰
│
├── public/               # 정적 파일 (이미지, 아이콘 등)
│
└── .storybook/           # Storybook 설정
```

## 🎨 디자인 시스템

이 프로젝트는 체계적인 디자인 토큰 시스템을 사용합니다.

### Design Tokens

- **Colors** - 색상 팔레트 (Blue, Green, Red, Orange, Lime, Grey)
- **Typography** - 타이포그래피 스타일
- **Spacing** - 간격 시스템

좌측 사이드바에서 `Design Tokens` 메뉴를 확인하세요!

## 🔧 개발 가이드

### 새 컴포넌트 만들기

1. `components/` 폴더에 새 컴포넌트 파일 생성
2. TypeScript와 React 19를 사용하여 컴포넌트 작성
3. Tailwind CSS 또는 Ant Design 컴포넌트 활용
4. `stories/components/` 폴더에 스토리 파일 추가

### 스타일링 가이드

```tsx
// Tailwind CSS 사용 예시
<div className="flex items-center gap-4 p-6">
  <span className="text-sm font-medium">Hello</span>
</div>

// Ant Design 컴포넌트 사용 예시
import { Button } from 'antd';

<Button type="primary">클릭</Button>
```
